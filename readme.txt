1/ sau khi clone project về vào folder gymweb thực hiện lệnh npm i để tải cái module cần thiết(cần có nodejs)(cd GymWeb && npm i)
2/ nếu có lỗi báo tìm tên module và thực hiện lệnh (npm i *tênmodule*)
3/ chạy server backend :sau khi đầy đủ module vào folder /src/backend/ thực hiện lệnh node server.js (cd /src/Backend && node server.js)
4/ chạy server frontend: ở folder source chính Gymweb npm start web sẽ chạy ở port 3000
